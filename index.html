About this project

If you are an electronics geek sometimes it is hard to remember the value of the resistor (Just like me). Instead of manually calculating let this web application do the hardwork for you. Currently it only supports 4 band resistor color 5 & 6 band will be added in later updates.

Just select the color in the proper sequence & get the correct value of the resistor that's it.
